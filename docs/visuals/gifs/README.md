## GIF Recordings

Interactive terminal recordings are auto-generated using [VHS](https://github.com/charmbracelet/vhs).

| Recording                         | Description                    |
|-----------------------------------|--------------------------------|
| `quickstart.gif`                  | Quick overview of osm commands |
| `super-document-visual.gif`       | Super-Document visual TUI mode |
| `super-document-shell.gif`        | Super-Document shell/REPL mode |
| `super-document-interop.gif`      | Mode switching demonstration   |
| `code-review.gif`                 | Code review workflow           |
| `prompt-flow.gif`                 | Two-step prompt builder        |
| `goal.gif`                        | Goal-based workflows           |
| `super-document-visual-light.gif` | Light theme variant            |

### Regenerating Recordings

```sh
make generate-tapes-and-gifs
```

Requires VHS to be installed:

```sh
brew install vhs # macOS
# or see https://github.com/charmbracelet/vhs#installation
```
